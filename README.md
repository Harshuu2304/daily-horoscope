# Initial-Based Daily Horoscope Web App

This is a mini web project developed as part of college learning.
The application predicts daily horoscope messages based on the
user's name initial using JavaScript switch-case logic.

## Features
- Simple and clean UI
- Initial-based horoscope logic
- Daily-changing messages
- Hosted using GitHub Pages

## Technologies Used
- HTML
- CSS
- JavaScript
- GitHub Pages
